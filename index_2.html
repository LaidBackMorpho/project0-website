What Programming Means to Me

To me, programming is more than just typing text on a screen and hoping it will work flawlessly each time it runs. Don't get me wrong, it's great when that does happen, but development is so much more. It's a beautiful process of problem-solving, learning about the particular issue trying to be dealt with and how to rectify all the nuances that come with it. By the time a program is finished, the person or team behind it will understand nearly all of the facets that the issue has to offer. After all, a successful program is a problem solved, no matter the size or impact.

The Programming Languages I Know

So far, the programming languages I've learned are Lua and Python. About six years ago I started learning Lua. I taught myself by looking at the source code of what others had created. Five years, two hiatuses, and one decision about what I want to do with my life later, I learned Python in college.

Notable Programs I've Made
